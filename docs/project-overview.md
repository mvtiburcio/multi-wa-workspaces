# Visão do Projeto

## Contexto
Objetivo de construir um app interno para operar múltiplas sessões de WhatsApp Web via QR Code, usando conceito de Workspaces independentes.

## Problema
Equipe precisa alternar entre múltiplas contas, com isolamento de sessão e experiência operacional mais organizada.

## Resultado esperado
- Workspaces independentes;
- onboarding por QR;
- persistência de sessão por workspace;
- gestão unificada no shell nativo.

## Fora de escopo neste momento
- backend de produção;
- integrações via API oficial de mensagens;
- publicação comercial em lojas.
